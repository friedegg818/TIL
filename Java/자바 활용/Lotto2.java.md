### Lotto 프로그램 
- 1~45 사이의 숫자 6개 + 보너스 숫자 1개 추첨 
- 로또 등수 
  
     1등 6개 일치 
     2등 5개 + 보너스번호 일치 
     3등 5개 일치 
     4등 4개 일치 
     5등 3개 일치 

Lotto.java

- 6개의 번호를 담을 수 있는 ArrayList 가 존재한다.

- 6개의 정수를 담아야 하기 때문에 기본 타입인 int 타입을 객체로 만들어서 ArrayList 에 추가해야 한다.
