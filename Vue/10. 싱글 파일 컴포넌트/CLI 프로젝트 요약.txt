vue-cli : 명령어 보조 도구(CLI)로 생성한 프로젝트 

npm run serve <- node package manager 를 이용한 실행 명령어
package.json 파일로 가면 명령어가 실행어로 정리 되어 있음 


public > index.html

main.js 의 내용을 통해서 app 이나 appHeader의 내용들이 들어갈 수 있다
