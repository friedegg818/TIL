### CLI 2.X 와 3.X 의 차이점 

- Vue CLI 2.x
  - vue **init** '프로젝트 템플릿 유형' '프로젝트 폴더 위치'
  - vue **init** webpack-simple '프로젝트 폴더 위치'

- Vue CLI 3.x
  - vue **create** '프로젝트 폴더 위치'
