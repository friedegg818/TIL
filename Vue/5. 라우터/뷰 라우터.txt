뷰 라우터
뷰 라이브러리를 이용하여 싱글 페이지 애플리케이션을 구현 할 때 사용하는 라이브러리 
(페이지 이동과 관련된 기능)

설치 
CDN 방식 

뷰 인스턴스에 라우터 주입 

라우터 옵션
routes : 라우팅 할 URL 과 컴포넌트 값 지정 
mode   : URL의 해쉬 값 제거 속성 

router-view
해당하는 url로 갔을 때 해당하는 컴포넌트가 가지고 있는 내용이 나타남 
브라우저 주소 창에서 URL이 변경되면, 앞에서 정의한 routes 속성에 따라 해당 컴포넌트로 뿌려지는 지점 

router-link 
웹 페이지에서 페이지 이동을 할 때 보통 사용자가 url을 다 쳐서 이동하지 않으므로 
클릭해서 페이지를 이동 할 수 있게 하는 것
